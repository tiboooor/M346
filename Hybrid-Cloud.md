# Hybrid Cloud

Eine Hybrid CLoud ist ein Cloud Computing-Modell, das eine Kombination aus einer Private Cloud und einer Public Cloud verwendet.
Diese zusammen geben eine flexible Mischung aus Cloud Computing-Services.
